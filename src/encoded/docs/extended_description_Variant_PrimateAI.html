PrimateAI predicts pathogenicity for missense variants based on comparison to  primate sequences and variants using a deep learning approach. Scores range  from 0-1 with scores above 0.803 indicating a pathogenic prediction. Predictions  can be one of "D" for damaging (>0.803) or "T" for tolerated (<0.803).  The rankscore is a value between 0-1 indicating where this variant's score falls  compared to all other variants with PrimateAI scores. For details see the  publication.
